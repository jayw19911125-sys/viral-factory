## 一、底層邏輯：為什麼影片會爆？

### 1.1 演算法的本質
平台（TikTok / IG Reels / Meta Ads）只做一件事：**最大化用戶停留時間**。
因此它獎勵的是：
- **3秒留存率**（Hook Rate）：前3秒沒留住 = 永遠不會被推
- **完播率**（Completion Rate）：看完才算有效觀看
- **重複播放**（Replay）：重看是最強的正向信號
- **互動密度**：留言 > 分享 > 收藏 > 點讚（權重由高到低）

### 1.2 爆款的三個層次
```
層次1：停下來看（Hook 成功）→ 3秒留存率 > 65%
層次2：看完（Body 成功）→ 完播率 > 40%
層次3：分享/收藏（Value 成功）→ 觀眾認為「這個值得傳給別人」
```

## 二、視覺錘 × 語言釘 理論（品牌定位核心）

### 2.1 定義
- **語言釘（Verbal Nail）**：一句話說清楚你是誰、你解決什麼問題。先確定，是策略層。
  - 例：「30分鐘到府，高雄最快的水電師傅」
  - 例：「不油膩的台式便當，每天都能吃的健康快餐」
- **視覺錘（Visual Hammer）**：把語言釘敲進觀眾腦袋的視覺符號。後確定，是執行層。
  - 例：師傅到府的計時器畫面
  - 例：便當打開的瞬間特寫 + 蒸氣

### 2.2 在短影音中的應用
| 元素 | 短影音中的位置 | 作用 |
|------|--------------|------|
| 視覺錘 | 前0-1秒 | 讓人停下來 |
| 語言釘 | 前1-3秒（字幕或口白） | 讓人知道為什麼要繼續看 |
| 人設定位 | 貫穿全片 | 讓人記住「這是誰」 |

## 三、鉤子（Hook）設計系統

### 3.1 五大鉤子類型

| 類型 | 公式 | 心理機制 | 最適產業 |
|------|------|---------|---------|
| **大膽宣言型** | 「[反常識結論]，因為[意外原因]」 | 認知衝突 | 教育/自媒體/創業 |
| **問題型** | 「你有沒有遇過[精準痛點]？」 | 好奇缺口 | 服務業/美容/餐飲 |
| **前後對比型** | 先秀結果，再說過程 | 社會證明 | 美容/電商/健身 |
| **好奇缺口型** | 「大多數人不知道的[秘密/方法]」 | Zeigarnik效應 | 所有產業 |
| **痛點激化型** | 「你還在[舊做法]？難怪[壞結果]」 | 情緒共鳴 | 廣告投放/B2B/服務業 |

### 3.2 各產業最佳鉤子配方

#### 🍜 餐飲業
**最有效鉤子**：視覺衝擊型（食物特寫）+ 好奇缺口型
```
公式1：[食物最誘人的畫面] + 「這個[食物名]為什麼讓人排隊3小時？」
公式2：「[地點]藏了一家[特色]的店，在地人才知道」
公式3：[製作過程特寫] + 「老闆說這個秘方從不外傳」
```
**視覺錘**：食物出爐/切開/醬汁流下的瞬間（ASMR感）
**語言釘**：「[城市]最[形容詞]的[食物]」
**人設定位**：在地達人 / 老饕 / 廚師本人

#### 💄 美容/美妝業
**最有效鉤子**：前後對比型 + 問題型
```
公式1：[Before畫面] + 「用了這個之後...」 + [After畫面]
公式2：「皮膚科醫師不會告訴你的[保養秘密]」
公式3：「[年齡]歲看起來像[更年輕]歲的原因只有一個」
```
**視覺錘**：肌膚對比、產品使用中的特寫、真實素顏
**語言釘**：「[時間]看到效果，不然退費」
**人設定位**：素人見證者 / 美容師 / 皮膚科醫師

#### 🛒 電商（實體商品）
**最有效鉤子**：前後對比型 + 大膽宣言型
```
公式1：「我花了[金額]買了[產品]，值不值得？」（開箱型）
公式2：「[競品]賣[高價]，這個只要[低價]，差在哪？」
公式3：「[使用場景痛點]？這個東西解決了我[時間]的困擾」
```
**視覺錘**：開箱瞬間 / 使用前後對比 / 產品特寫
**語言釘**：「[具體效果]，[具體時間]，[具體保證]」
**人設定位**：真實用戶 / 測評達人 / 生活達人

#### 🏢 服務業（B2B/專業服務）
**最有效鉤子**：痛點激化型 + 好奇缺口型
```
公式1：「你的[廣告/網站/社群]為什麼沒有效？」
公式2：「[行業]老闆最常犯的[數字]個錯誤」
公式3：「我幫[客戶類型]在[時間]內達到[結果]，方法是...」
```
**視覺錘**：數據截圖 / 成果對比 / 真實案例畫面
**語言釘**：「[具體成果]，[具體時間]，[具體客戶類型]」
**人設定位**：行業專家 / 成功案例當事人 / 顧問

## 四、影片結構公式（黃金三段式）

### 4.1 標準結構
```
[0-3秒] HOOK（鉤子）：停下來
[3-45秒] BODY（主體）：留下來
[45-60秒] CTA（行動呼籲）：做什麼
```

### 4.2 各段詳細設計

#### Hook（前3秒）—— 決定生死
- **視覺層**：畫面要有動態、對比、或意外感
- **聽覺層**：前0.5秒就要有聲音（不能靜音開場）
- **文字層**：字幕要在畫面出現前0.5秒就顯示
- **禁忌**：Logo動畫、自我介紹、「大家好我是XXX」

#### Body（中段）—— 留存關鍵
- 每3-5秒要有一個「小鉤子」（轉折/新資訊/視覺變化）
- 節奏要踩音樂節拍
- 資訊要碎片化（條列 > 長段落）
- 製造「下一秒會怎樣」的期待感

#### CTA（結尾）—— 轉化關鍵
- 有機內容：「留言[關鍵字]領取」「標記一個需要看這個的朋友」
- 廣告內容：「點擊連結」「私訊我」「限時優惠」
- 禁忌：「謝謝觀看」「記得訂閱」（太弱）

## 五、人設定位系統

### 5.1 六種人設類型
| 人設類型 | 核心特質 | 最適產業 | 視覺錘 |
|---------|---------|---------|-------|
| **專家型** | 權威、知識、可信度 | 醫美/法律/財務/教育 | 白袍/辦公室/證書 |
| **素人見證型** | 真實、親切、可複製 | 電商/美容/健身 | 素顏/居家環境/真實反應 |
| **創業者型** | 奮鬥、誠實、共鳴 | 服務業/自媒體/B2B | 工作現場/幕後/數據 |
| **在地達人型** | 熟悉、信任、獨家 | 餐飲/旅遊/生活 | 在地場景/老饕感 |
| **生活達人型** | 品味、靈感、嚮往 | 家居/時尚/美食 | 美好生活畫面 |
| **挑戰者型** | 顛覆、對比、刺激 | 所有產業 | 對比畫面/測試現場 |

### 5.2 人設 × 產業 × 鉤子 配方表
| 產業 | 推薦人設 | 最強鉤子類型 | 視覺錘 | 語言釘範例 |
|------|---------|------------|-------|----------|
| 餐飲 | 在地達人/廚師本人 | 視覺衝擊+好奇缺口 | 食物特寫/製作過程 | 「高雄人才知道的隱藏版」 |
| 美容美甲 | 素人見證/美容師 | 前後對比+問題型 | 肌膚對比/操作特寫 | 「[時間]改變你的[部位]」 |
| 電商服飾 | 生活達人/素人 | 前後對比+開箱 | 穿搭對比/開箱瞬間 | 「[價格]穿出[效果]」 |
| 電商3C | 測評達人/素人 | 大膽宣言+測試 | 產品測試/數據對比 | 「[競品]vs[產品]，差在這」 |
| 健身/運動 | 素人見證/教練 | 前後對比+挑戰 | 身材對比/訓練現場 | 「[時間]從[Before]到[After]」 |
| 服務業 | 創業者/專家 | 痛點激化+案例 | 成果截圖/客戶見證 | 「幫[客戶]在[時間]達到[結果]」 |
| 教育/課程 | 專家/學員見證 | 好奇缺口+問題型 | 學習成果/知識圖表 | 「學會這個，[問題]不再困擾你」 |

## 六、Meta 廣告頂尖投放邏輯

### 6.1 2026 Andromeda 演算法後的新規則
- **素材 > 受眾**：系統自己找人，你只要給夠好的素材
- **廣告組合放5-10支**素材，讓系統自動分配
- **每2-4週換素材**：同一批受眾看超過3-4次就會疲勞
- **Reels格式優先**：CPM比Feed低15-30%

### 6.2 各產業廣告指標基準（台灣市場參考）
| 產業 | 目標ROAS | CPA目標 | 最佳素材格式 | 鉤子類型 |
|------|---------|---------|------------|---------|
| 餐飲（導客流） | 不看ROAS，看CPA | NT$50-150/來客 | 短影片9:16 | 視覺衝擊+限時優惠 |
| 美容/美甲 | 2.5-4x | NT$200-500 | UGC短影片 | 前後對比+問題型 |
| 電商（服飾/家居） | 2-3x | NT$300-800 | 輪播+短影片 | 開箱+前後對比 |
| 電商（3C/工具） | 1.8-2.5x | NT$500-1500 | 示範影片 | 大膽宣言+測試 |
| 服務業（B2B） | 不看ROAS | NT$500-2000/名單 | 短影片+圖文 | 痛點激化+案例 |

### 6.3 高成效廣告素材的無效過濾標準
以下特徵代表素材**無效**，應立即淘汰：
- 前3秒是Logo動畫或自我介紹
- 靜音播放無法理解內容（85%用戶靜音）
- 圖片上文字超過10個字
- 素材與著陸頁承諾不一致
- 同一素材跑超過4週未更新
- CTR < 0.8%（冷受眾）
- 3秒留存率 < 30%

### 6.4 廣告投放成效判斷SOP
```
Day 1-2：只看CPM和3秒留存率（素材能不能被看到）
Day 3-5：看CTR和點擊成本（素材能不能吸引點擊）
Day 5-7：看CPA和ROAS（素材能不能轉換）
Week 2+：看頻率（Frequency > 3.5 = 素材疲勞，準備換）
```

## 七、無效影片過濾標準（拆解前先篩）

### 7.1 有機短影音（TikTok/IG Reels）
**有效影片特徵（至少符合3項）**：
- 播放量 > 帳號平均播放量的3倍以上
- 留言中有「在哪裡買」「怎麼做」「求連結」等購買意圖
- 分享數 > 點讚數的10%
- 影片發布7天內就達到高播放（快速爆發）
- 有明確的鉤子設計（前3秒有視覺或語言鉤子）

**無效影片特徵（有任一項就排除）**：
- 純粹搞笑/娛樂，無商業參考價值
- 播放量高但留言都是「哈哈哈」（娛樂型，非導購型）
- 依賴特殊資源（大明星/重大事件），無法複製
- 前3秒是自我介紹或Logo
- 影片品質差（畫面模糊/聲音不清）

### 7.2 Meta廣告素材
**高成效廣告特徵**：
- 投放超過30天（長壽 = 高成效指標）
- 有明確的鉤子 + 痛點 + 解方 + CTA結構
- 使用UGC風格（真人出鏡，非精緻廣告感）
- 有具體數字（「3天」「50%」「NT$299」）
- 靜音也能理解核心訊息（字幕完整）
